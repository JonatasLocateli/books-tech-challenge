# Integração com Machine Learning

## Objetivo
Permitir que cientistas de dados e engenheiros de ML consumam dados de forma simples e estruturada.

## Endpoints ML-Ready
- GET /api/v1/ml/features
- GET /api/v1/ml/training-data
- POST /api/v1/ml/predictions

## Casos de Uso
- Treinamento de modelos de recomendação
- Análise exploratória de dados
- Feature engineering

## Escalabilidade
- Possível integração com pipelines de ML (Airflow, MLflow)
- Migração futura para Data Warehouse
